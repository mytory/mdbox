# Mytory MDBox — 도메인 용어집

## 핵심 정체성

| 용어 | 정의 |
|---|---|
| **Mytory MDBox** | 파일을 드롭하면 Markdown으로 변환해 주는 크로스플랫폼 데스크톱 앱. 묻지도 따지지도 않고 변환한다. |
| **Markitdown** | Microsoft의 오픈소스 Python 라이브러리. 다양한 파일 포맷을 Markdown으로 변환. 앱의 핵심 변환 엔진. |

## 핵심 개념

| 용어 | 정의 |
|---|---|
| **드롭앤던이 (Drop-and-done)** | 파일을 앱에 드롭하기만 하면 즉시 Markdown이 준비되는 UX 원칙. 중간 다이얼로그, 설정 묻는 것 없음. |

## 출력 모드

| 용어 | 정의 |
|---|---|
| **Sidecar 모드 (기본)** | 원본 파일과 같은 폴더에 `.md` 파일 생성. 예: `report.docx` 드롭 → 같은 폴더에 `report.md` 생성. |
| **클립보드 모드** | 변환된 Markdown을 시스템 클립보드에 자동 복사. 여러 파일 순차 변환 시 마지막 파일의 Markdown이 클립보드에 남음. |
| **사용자 지정 출력 모드** | 설정에서 지정한 출력 폴더에 `.md` 파일 생성. Sidecar 모드 대신 사용. |
| **Sidecar + 클립보드 모드** | Sidecar 모드와 클립보드 모드를 동시 활성화. 두 방식 모두 동작. |

## UI 레이아웃

| 용어 | 정의 |
|---|---|
| **상단 작업 큐** | 화면 상단에 고정(sticky)된 작업 큐. 변환 진행 상황을 항상 볼 수 있음. mytory-video-tools와 동일 패턴. |
| **중앙 드롭존** | 화면 중앙의 드롭존. 파일을 드롭하거나 클릭하여 파일 선택 가능. 지원 포맷 목록 표시. |
| **설정 아이콘 (⚙️)** | 우측 상단 설정 아이콘. 모달로 출력 모드, 폴더 지정 등을 설정. |
| **다크 테마** | mytory-video-tools와 동일한 다크 테마 계승. |

## 드롭 동작

| 용어 | 정의 |
|---|---|
| **단일 드롭** | 파일 하나를 드롭. 즉시 변환 후 .md 생성. |
| **다중 드롭** | 여러 파일을 동시 드롭. 내부 큐에 등록되어 순차 변환. |
| **폴더 드롭** | 폴더를 드롭하면 내부의 지원 포맷 파일을 재귀적으로 찾아 모두 변환. 폴더 구조는 유지되지 않고 모든 .md 파일이 출력 대상에 평탄화됨. |
| **변환 후 원본 삭제** | ❌ 지원하지 않음. |
| **변환 큐** | 다중/폴더 드롭 시 파일이 순차적으로 변환되는 대기열. 진행률을 실시간 표시. mytory-video-tools의 Task Queue 패턴 계승. |

## 설정

| 용어 | 정의 |
|---|---|
| **출력 모드 선택** | Sidecar (기본) / 사용자 지정 출력 / Sidecar+클립보드 / 사용자 지정+클립보드 중 선택. |
| **출력 폴더** | 사용자 지정 출력 모드 사용 시 저장 경로 지정. |
| **LLM 제공자** | 이미지 캡셔닝용 LLM 설정. OpenAI 호환 API이면 무엇이든 가능. |
| **LLM 엔드포인트** | OpenAI API 키 또는 Ollama/LM Studio 등 로컬 엔드포인트 URL. |
| **LLM 모델** | 사용할 모델명 (예: gpt-4o, llama3.2-vision). |

## 향후 과제 / 범위 밖

| 용어 | 정의 |
|---|---|
| **.doc (구형 Word)** | Binary OLE 포맷. 현재 markitdown이 미지원. 향후 macOS `textutil` 또는 bundled `antiword` 바이너리로 대응 가능. |
| **.hwp / .hwpx** | 한글 워드프로세서 포맷. 현재 markitdown이 미지원. 별도 컨버터 개발 또는 라이브러리 연동 필요. |
| **컨텍스트 메뉴 연동** | 파인더/탐색기에서 우클릭 → "Markdown으로 변환" 메뉴. OS별 구현 필요. v1.0 범위 밖. |
| **파일 연결 (Open with)** | ❌ 지원하지 않음. 컨텍스트 메뉴로 대체. |
