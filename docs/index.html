One drop, done

The simplest file-to-Markdown converter you'll ever use. Drag, drop, and walk away. No dialogs, no settings, no friction.

Drop-and-Done

The entire UX: drop a file, get Markdown. No prompts, no settings dialogs, no "how would you like to save this?" Just the result, ready to use. Works with single files, multiple files, and entire folders.

  • Single files
  • Batch processing
  • Folder recursion

Office Documents

Convert DOCX, PPTX, and XLSX to clean Markdown. Tables become Markdown tables, slides become headings, spreadsheets become data tables.

PDF & Web

Extract text from PDF files and convert HTML pages to Markdown. Perfect for archiving web articles or extracting content from documents.

Code & Data

CSV becomes tables, JSON and XML become formatted code blocks, IPYNB notebooks become clean Markdown. Source code files are preserved verbatim.

EPUB & More

Convert EPUB ebooks, emails (.msg), ZIP archives, and plain text. If a format can hold text, MDBox can extract it as Markdown.

LLM-Ready Output

Markdown is the native language of LLMs. MDBox produces clean, well-structured Markdown that you can paste directly into ChatGPT, Claude, or any AI tool. Optionally connect your OpenAI-compatible endpoint for automatic image captioning.

The idea

Why another converter?

Every document converter asks questions. "Where should I save this?" "What format?" "Do you want to include images?" "Which options would you like?" — By the time you've answered, you could have done it manually.

Mytory MDBox does not ask questions. Drop a file onto the window, and the Markdown is already waiting for you beside the original. No dialogs, no settings, no interruptions. Just the result.

Under the hood, it uses Microsoft Markitdown — the same engine that powers AI document processing at scale. Every format, from DOCX to EPUB to PDF, is handled with production-grade accuracy. And since everything runs locally, your documents never leave your machine.
