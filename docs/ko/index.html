드롭 하나로 끝

가장 단순한 파일→Markdown 변환기. 드래그, 드롭, 끝. 다이얼로그도, 설정도, 마찰도 없습니다.

드롭앤던이

UX의 전부입니다: 파일을 드롭하면 Markdown이 나옵니다. 프롬프트도, 설정 다이얼로그도, "어디에 저장할까요?"도 없습니다. 그냥 결과물이 준비됩니다. 단일 파일, 여러 파일, 폴더 전체 모두 지원.

  • 단일 파일
  • 일괄 처리
  • 폴더 재귀

오피스 문서

DOCX, PPTX, XLSX를 깔끔한 Markdown으로 변환. 표는 Markdown 테이블로, 슬라이드는 제목으로, 스프레드시트는 데이터 테이블로.

PDF 및 웹

PDF 파일에서 텍스트를 추출하고 HTML 페이지를 Markdown으로 변환. 웹 문서 아카이빙이나 문서 내용 추출에 최적.

코드 및 데이터

CSV는 테이블로, JSON과 XML은 코드 블록으로, IPYNB 노트북은 깔끔한 Markdown으로. 소스 코드는 그대로 보존.

EPUB 등

EPUB 전자책, 이메일(.msg), ZIP 아카이브, 일반 텍스트 등 다양한 포맷 지원. 텍스트를 담을 수 있는 형식이라면 MDBox가 Markdown으로 추출합니다.

LLM 친화적 출력

Markdown은 LLM의 모국어입니다. MDBox는 ChatGPT, Claude 등 모든 AI 도구에 바로 붙여넣을 수 있는 깔끔한 Markdown을 생성합니다. 선택적으로 OpenAI 호환 엔드포인트를 연결하면 이미지 자동 캡셔닝도 가능.

  • AI 최적화
  • 이미지 캡셔닝
  • OpenAI / Ollama

또 변환기?

모든 문서 변환기는 질문을 합니다. "어디에 저장할까요?" "어떤 형식으로?" "이미지를 포함할까요?" "어떤 옵션을 사용하시겠습니까?" — 이 질문에 답하는 동안 수동으로 해치웠을 시간입니다.

Mytory MDBox는 질문하지 않습니다. 파일을 창에 드롭하면 Markdown이 원본 옆에 이미 기다리고 있습니다. 다이얼로그도, 설정도, 방해도 없습니다. 그냥 결과물입니다.

내부적으로는 Microsoft Markitdown을 사용합니다 — AI 문서 처리를 위해 만들어진 바로 그 엔진입니다. DOCX부터 EPUB, PDF까지 모든 형식을 프로덕션 수준의 정확도로 처리합니다. 모든 작업은 로컬에서 실행되므로, 문서가 내 컴퓨터를 떠나지 않습니다.

다이얼로그 없음 100% 오프라인 Microsoft Markitdown 프라이버시 우선 오픈소스
